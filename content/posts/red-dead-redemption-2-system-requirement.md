## Red Dead Redemption 2 System Requirements:

|   |  |
| ----- | --- |
| **CPU:**   | 	Intel Core i5-2500K / AMD FX-6300  |
| **RAM:**   | 	8 GB RAM |
| **GPU:**   | 		Nvidia GeForce GTX 770 2GB / AMD Radeon R9 280 3GB  |
| **OS:**    | 	Windows 7 - Service Pack 1 (6.1.7601), 64-bit  |
| **STO:**   | 	150 GB available space |
